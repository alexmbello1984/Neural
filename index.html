Neural Framework

Neural Framework es un marco de trabajo de aplicaciones en PHP separando la lógica de negocio, consultas a bases de datos y la presentación de los resultados.

Neural Framework es un marco de trabajo en PHP el cual esta constituido sobre módulos de aplicación el cual me permite tener multiples aplicaciones en si mismo.

Neural Framework puede utilizar las siguientes opciones:

  1. Generar Validación de Formularios a través de JQuery.
  2. Administrar Scripts de forma persistente para el manejo de las vistas.
  3. Algunas funciones AJAX para manejo de vistas como administración de contenido.
  4. Centralización de configuraciones de las aplicaciones haciendo más sencillo mantenerlas.
  5. Generar encriptaciones de dos vias de forma sencilla para proteger los datos más importantes.
  6. Una opción de Cache Simple de resultados o datos correspondientes a consultas ó aplicación.
